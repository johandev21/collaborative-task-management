# Tech Stack Definition: NestJS, Drizzle ORM, React, Vite & TanStack

We established the technology stack for the Collaborative Task Management Platform:

- **Backend**: NestJS framework providing modular structure, dependency injection, and clean Hexagonal Architecture ports/adapters.
- **Database & ORM**: PostgreSQL with Drizzle ORM for lightweight, performant, and type-safe schema definitions and queries.
- **Validation**: Zod for end-to-end type validation on HTTP endpoints, WebSocket payloads, and domain inputs.
- **Frontend**: React + Vite for fast build tooling and modern component rendering.
- **Routing**: TanStack Router for compile-time type-safe routing and search parameter handling.
- **State & Data Fetching**: TanStack Query (React Query) for server-state caching, optimistic updates, and cache invalidation.
- **UI Components**: shadcn/ui primitives with Tailwind CSS for modern, responsive, and accessible user interfaces.
